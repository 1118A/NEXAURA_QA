RAG Pipeline Architecture

1. Clone & Standardize

Clones or pulls the requested branch, verifies directories, and filters out non-code assets, ensuring clean context loading.

2. Tree-sitter AST Chunker

Parses syntax trees using Tree-sitter bindings for Python, JS, and TS, isolating functions, classes, and methods as semantic blocks.

3. Citation Embedding

Embeds items into a high-dimensional vector space using bge-small-en-v1.5, tracking line numbers and scopes for precise traceback reference.

4. Cosine Search & Groq RAG

Executes fast cosine checks, filters weak embeddings via thresholds, and leverages 70B Groq inference to generate halluncination-free code answers.
